# MotoBrain AI — Frontend

Dashboard web del proyecto [MotoBrain AI](../README.md): panel para talleres de motos (inventario, clientes, servicios, diagnóstico IA y analítica).

## Stack

- Next.js 14 (App Router) · TypeScript · TailwindCSS
- TanStack Query · Zustand · React Hook Form · Zod
- Recharts · Lucide · Sonner

## Desarrollo

```bash
npm install
cp .env.local.example .env.local
npm run dev
```

Asegúrate de tener la **API** en marcha (`npm run dev` en la raíz del monorepo) y `NEXT_PUBLIC_API_URL` apuntando a ella.

Documentación completa del producto y despliegue: **[README principal](../README.md)**.
